## Project setup

### Clones repository

```
git clone https://github.com/manankarnik/contest.git
```

### Installs dependencies

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
